# Sismógrafo – Estimación de Riesgo Sísmico

Desarrollo de un prototipo de plataforma digital que estima el riesgo sísmico
en Oaxaca

## Objetivo
Desarrollar un sistema que integre bases de datos, fórmulas comprobadas y una interfaz
para clasificar niveles de riesgo sísmico y simular escenarios de afectación.

## Tecnologías
- Python
- PostgreSQL
- React
- HTML, CSS, JavaScript
- Git & GitHub

## Alcance
Este sistema no predice sismos. Realiza estimaciones de riesgo basadas en datos históricos.
